# Tower Defense Game

A Tower Defense game developed in C# using Windows Forms as part of a university programming project. The project focuses on applying Object-Oriented Programming (OOP) concepts through the implementation of towers, enemies, game management, and upgrade mechanisms.

## Project Overview

The game simulates a tower defense environment where different tower types attack enemies moving along a predefined path. The project was designed to demonstrate software design principles and object-oriented programming techniques rather than commercial game development.

## Implemented Components

### Enemy System

* Enemy representation and movement logic.
* Path-based enemy progression.
* Enemy management within the game cycle.

### Tower System

* Multiple tower types:

  * Arrow Tower (OkKulesi)
  * Cannon Tower (TopKulesi)
  * Magic Tower (BuyuKulesi)
* Attack behavior implementation.
* Tower-specific functionality and characteristics.

### Game Management

* Centralized game control through the GameManager class.
* Coordination of towers, enemies, and gameplay events.

### Upgrade and Attack Mechanisms

* Attack functionality implemented through interfaces.
* Upgrade functionality implemented through interfaces.
* Reusable and extensible design for future tower types.
